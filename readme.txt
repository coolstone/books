一、设立技术资料库
面向公司开发和实施人员的技术培训和自我学习，在\\zsoftserver设立的技术资料库主要内容规划如下：
1、基础框架：秉政基础框架所采用技术的电子书、文档、PPT、视频等资料。
2、开发入门：开发人员应掌握基本技能的电子书、文档、PPT、视频等资料。
3、部署实施：系统部署实施应掌握技能的电子书、文档、PPT、视频等资料。
4、技术方向：未来研发方向拟采用技术的电子书、文档、PPT、视频等资料。
 
二、技术资料库访问
\\192.168.0.254\技术资料库   
公司所有人均可访问，研发部门经理按需通知开发人员不同阶段的阅读重点。
 
三、技术资料库管理
由技术总监、研发部经理、程序经理对技术资料库的目录和文件进行管理。
